MEMO
- `func!`の呼び出し方のfuncはマクロ
- 高階関数の呼び方はRuby的：`.map(|i| i*i)`（個人的にはこれ、タイピングしにくくて、可読性的にも見づらくて嫌いなんだよな、、、）
- `mut`は「変数」につく属性。letみたいな宣言子につくわけでない。`let mut x = 2;`
- `&[]` : 配列への参照。sliceとも呼ぶ。
- `&str`：文字列への参照。別名なし。
- mutなrefは２つ以上借用できない。
- immuなrefはmultiにcreate可能
- Mut/immuでrefをそれぞれ持てれない。
